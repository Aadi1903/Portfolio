# Personal Portfolio Website

## About
A clean, modern, and fully responsive personal portfolio website built to showcase my software development skills and my growing journey in Cloud and DevOps. This platform serves as a central hub for my technical expertise, professional timeline, certifications, and featured projects.

## Features
- Responsive design for desktop, tablet, and mobile devices
- Projects showcase highlighting technical work
- Skills section mapping technical proficiencies
- Certifications section displaying achievements
- Journey (timeline) showing academic and professional milestones
- Contact section with a functional form

## Sections Overview
- **Home:** Hero section with a quick introduction
- **Skills:** Technical toolkit (Cloud & DevOps, Programming, Tools)
- **Projects:** Detailed project cards with descriptions
- **Certifications:** Verified credentials
- **Journey:** Timeline of academic and professional growth
- **About Me:** Background summary and social links
- **Contact:** Direct communication form

## Tech Stack
- Frontend: React, JavaScript (ES6+)
- Styling: Tailwind CSS
- Icons: Lucide React

## Setup Instructions
To run this project locally, follow these steps in your terminal:

```bash
git clone https://github.com/Aadi1903/Portfolio.git
cd vite-project
npm install
npm run dev
```
